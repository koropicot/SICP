＃問題1.9

##定義
```scheme
(define (inc x) (+ x 1))
(define (dec x) (- x 1))

(define (plus-1 a b)
  (if (= a 0)
      b
      (inc (plus-1 (dec a) b))))


(define (plus-2 a b)
  (if (= a 0)
      b
      (plus-2 (dec a) (inc b))))
```

##plus-1
```
(plus-1 4 5)
(inc (plus-1 3 5))
(inc (inc (plus-1 2 5)))
(inc (inc (inc (plus-1 1 5))))
(inc (inc (inc (inc (plus-1 0 5)))))
(inc (inc (inc (inc 5))))
(inc (inc (inc 6)))
(inc (inc 7))
(inc 8)
9
```
再帰的プロセス

(plus-2 4 5)
