#問題1.10

```scheme
;Ackermann
(define (A x y)
  (cond ((= y 0) 0)
        ((= x 0) (* 2 y))
        ((= y 1) 2)
        (else (A (- x 1)
                 (A x (- y 1))))))
```

```scheme
(A 1 10)
(A 0 (A 1 9))
(A 0 (A 0 (A 1 8)))
(A 0 (A 0 (A 0 (A 1 7))))
```
人間じゃ辛い

```scheme
gosh> (A 1 10)
1024
gosh> (A 2 4)
65536
gosh> (A 3 3)
65536
```

---

```scheme
(f n)
(A 0 n)
(* 2 n)
```
fは2nを計算する。

---
```scheme
(g n)
(A 1 n)
(A 0 (A 1 (- n 1)))
...
(A 0 (A 0 ..[n-1個].. (A 1 1)...))
(A 0 (A 0 ..[n-1個].. 2...))
(* 2 (* 2 ..[n-1個].. 2...))
2*2* ..[n個].. *2
```
gは2^nを計算する。

---
```scheme
(h n)
(A 2 n)
(A 1 (A 2 (- n 1)))
...
(A 1 (A 1 ..[n-1個].. (A 2 1)...))
(A 1 (A 1 ..[n-1個].. 2...))
(A 1 n)は上より2^nなので
2^2^ ..[n個].. ^2
```
hは2↑↑nを計算する。
